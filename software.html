Software

Giotto
– Integrative data analysis and visualization toolkit for spatial transcriptomic data

Official Website

smfishHmrf
– A Python package for single-molecule FISH machine learning analyses. Includes SVM-based cell type mapping
from scRNAseq to seqFISH and spatial pattern mining using hidden Markov random field.

R Code Python Code

CUT&RUNTools
– A customizable workflow for performing analyses of CUT&RUN data. Besides reads trimming, alignment, peak calling,
it includes de novo motif finding, motif footprinting, and transcription factor binding site predictions.

Bitbucket Code

SEEK
– Search-based exploration of expression compendium. A coexpressed gene search and visualization system.

Official Website

ModSEEK
– Model organism extension of SEEK. Covers transcriptome compendia in fly, worm, mouse, zebrafish, and yeast.
